# Web CATRACHO

Sitio institucional de la Cámara de Transporte de Carga de Honduras.

## Stack

React 19 + TypeScript 5 + Vite 7 + Bootstrap 5 (como capa de soporte para grid y utilidades).

## Comandos

```bash
npm install     # instalar dependencias
npm run dev     # servidor de desarrollo
npm run build   # build de producción (tsc + vite build)
npm run preview # servir el build compilado
```
